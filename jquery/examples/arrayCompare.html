This page targeted to depict the usage of jQuery plugin for Array Compare.. While working on one of project came across the problem where I need to compare 2 objects and were looking for one common plugin. With little search decided to write my own simple plugin for the same and thought to share the same with community. Here are the details, problem statement for this plugin
  1. Have 2 simple arrays, i.e., Array having elements of premitive data types
  2. No need to too extra logic for comparing
  3. Single level arrays, i.e., no nested arrays
So having these things in mind started writing code and here is the plugin. Lets evaluate this plugin with some examples.
1. Positive comparison -- Lets take 2 Arrays having same elements in different order with same length of childs 
var arr1 = ["first", 2, "three", 100], arr2 = [2, 100, "first", "three"];

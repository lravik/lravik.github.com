This page targeted to depict the usage of jQuery plugin for Objects Compare.. Problem statement for this plugin
  1. Need to compare 2 objects
  2. Objects need to be same size to get compared. We won't compare one object partial with other object
  3. Object child elements need not to be in same sequence, could be in any order
  4. Expecting objects are having childs in key/value pairs
So having these things in mind started writing code and here is the plugin. Lets evaluate this plugin with some examples.
Positive comparison -- Lets take 2 Arrays having same elements in different order with same length of childs 
var obj1 = {"first": 1, "second": 2, "three": 4}, obj2 = {"first": 1, "three": 4, "second": 2};

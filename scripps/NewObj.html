New Object creation using JavaScript Function constructor

Object is one the premitive data type in JavaScript, every function we define is a Object. JavaScript is little different than conventional other Object oriented languages, it has flexible function definitions. It doesn't limit user to have functions with certain number of arguments. Eg. we could access any functions with dynamic number of arguments using "arguments" key object. So 2 functions having different set of arguments with same name will show errors in JavaScript. Now, as JavaScript doesn't restrict the developer in using functions with arguments, we could have default function with no arguments as constructor. So 

						var testFunc = function() {
							//Member variables
							var first, second;
							//Member functions
							var firstFunc = function() {
								return "First";
							};
						};
The function defined above could be called using default constructor using "new" operator for creating a new object. This is somewhat called constructor definition. 
						var test1 = new testFunc();
We could use some other design patterns too to achieve better OOPs structure. Few are: factory way, prototyping, decorators, etc.
